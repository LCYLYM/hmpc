## ⏳ 下一步：安装 Android SDK

### 方案 1: 安装 Android Studio（推荐）

1. **下载 Android Studio**
   - 官网: https://developer.android.com/studio
   - 或中文站: https://developer.android.google.cn/studio
   - 大小约 1GB

2. **安装步骤**
   - 运行安装程序
   - 选择 "Standard" 安装类型
   - 等待自动下载 Android SDK（约 3-5 GB）

3. **安装后配置**
   ```powershell
   # SDK 会自动安装到这个位置
   C:\Users\您的用户名\AppData\Local\Android\Sdk
   ```

4. **创建 local.properties 文件**
   安装完成后，在项目根目录创建 `local.properties` 文件：
   ```properties
   sdk.dir=C\:\\Users\\您的用户名\\AppData\\Local\\Android\\Sdk
   ```

### 方案 2: 仅安装命令行工具

如果不想安装 Android Studio IDE：

1. **下载 SDK Command Line Tools**
   - https://developer.android.com/studio#command-tools
   - 下载 Windows 版本

2. **解压到指定目录**
   ```powershell
   # 建议目录
   C:\Android\cmdline-tools
   ```

3. **使用 sdkmanager 安装必要组件**
   ```powershell
   cd C:\Android\cmdline-tools\bin
   .\sdkmanager.bat "platform-tools" "platforms;android-34" "build-tools;34.0.0"
   ```

4. **创建 local.properties**
   ```properties
   sdk.dir=C\:\\Android
   ```
